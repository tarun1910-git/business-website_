## How to Download Images

1. Go to https://unsplash.com or https://pexels.com
2. Search for each dish name above
3. Download the image
4. Rename it to the name listed above
5. Save to your `images/` folder

## Image Naming Convention
- Use lowercase names
- Use hyphens instead of spaces (e.g., `prime-ribeye.jpg`)
- File format: `.jpg` or `.png` (jpg recommended for better performance)

Once you download all images, the website will display them automatically!
